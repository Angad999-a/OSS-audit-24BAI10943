VLC Media Player — Open Source Software Study
📌 Overview

This repository contains an academic analysis of VLC Media Player as part of an Open Source Software study. The report explores the software’s origin, licensing, ecosystem, Linux deployment, and comparison with proprietary alternatives.

🎯 Objective

The goal of this project is to understand:

The philosophy of open source software
Licensing and software freedom
Linux deployment and package management
Community-driven development
Comparison with proprietary software
🧠 Origin Story

VLC Media Player was originally developed by students at École Centrale Paris. They needed a flexible and free tool capable of streaming multimedia over a university network. Existing software was either proprietary, limited, or lacked codec support. Instead of keeping their solution private, the developers released it as open source, allowing the global community to improve and expand it.

Four Freedoms
Freedom to run the program
Freedom to study the source code
Freedom to modify the software
Freedom to redistribute copies

VLC grants all four freedoms.

GPL vs MIT
GPL requires modified versions to remain open source
MIT allows closed-source redistribution
GPL protects community contributions

sudo apt install vlc -y

/usr/bin/vlc

Runs under normal logged-in user
Does not require root privileges

Update model:

Managed through package repositories
Security patches delivered via apt

✅ Verdict

VLC Media Player is recommended for real-world deployment due to its flexibility, codec support, and open development model.
